## Detect domain reconnaissance

Common native tools/commands utilized for domain reconnaissance include:
- `whoami /all`
- `wmic computersystem get domain`
- `net user /domain`
- `net group "Domain Admins" /domain`
- `arp -a`
- `nltest /domain_trusts`
